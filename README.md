# Shop by Season

**Live Site**: [shopbyseason.co](https://shopbyseason.co)

A curated fashion discovery platform for seasonal color analysis. Shop pieces matched to your personal color palette from top retailers including Zara, H&M, Mango, Next, and Mohito.

## About

Shop by Season helps you shop with confidence by hand-matching real, buyable fashion items to each of the 12 seasonal color palettes. No more guessing if a color suits you — every piece is curated for your exact season.

### Features

- 🎨 **12 Seasonal Palettes** - Complete color palettes for Light Spring, Warm Spring, Bright Spring, Light Summer, Cool Summer, Soft Summer, Soft Autumn, Warm Autumn, Deep Autumn, Bright Winter, Cool Winter, and Deep Winter
- 🛍️ **Curated Shopping** - Real products from popular retailers with direct purchase links
- 📊 **Google Sheets Integration** - Easy product management via spreadsheet (no coding required!)
- 🎯 **Smart Filtering** - Filter by category, color shade, and season
- 💰 **Sale Tracking** - Automatic SALE badges and strikethrough pricing
- 📱 **Responsive Design** - Works beautifully on all devices
- ⚡ **Fast & Lightweight** - Static site, no backend required

## Tech Stack

- Pure HTML, CSS, JavaScript (no frameworks)
- **Google Sheets** - Product data management (CSV feed)
- Hosted on GitHub Pages
- Custom domain via Cloudflare

## Color Analysis System

Based on the 12-season color analysis system that categorizes personal coloring into:
- **Springs** (warm, light to bright)
- **Summers** (cool, soft to light)
- **Autumns** (warm, muted to deep)
- **Winters** (cool, bright to deep)
